Automate Ahref File Download

2021-06-18

Automate href attribute download from a webpage

var list = document.getElementsByClassName("<enter classname here>")

Change HTMLCollection type to array

var array = Array.prototype.slice.call(list)

Loop over the map and download files one at a time. Note setTimeOut is added to wait until the file is downloaded.

array.map((elm, index) => {
  setTimeOut(() => {
    elm.click()
  })

}, 1000 * index)
